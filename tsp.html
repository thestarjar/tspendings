Welcome to The Stanley Parable Endings Guide! This guide is designed to help you get all the endings in the original game, while avoiding any spoilers aside from suggesting places to go. IF you say you are playing Ultra Deluxe, this will include the normal branches added to the base game in Ultra Deluxe, not any obvious new content. This guide is only for endings: it does not contain easter eggs or extras, only paths where either the game restarts or you are forced to do so manually.

It's recommended that you play The Stanley Parable blind and get all the endings you can find exploring on your own, then start on this guide afterwards. Do what you want, though!
